linux端开放失败的问题

发表于 2020-02-16 更新于 2020-05-27
本想上宝塔面板管理下网页的,发现打不开,扫描发现8888端口关闭,于是连上服务器,防火墙开放端口,并检测显示端口是开放的.但是端口扫描显示就是关闭的,且无法访问.

linux端口常用命令

firewall-cmd --zone=public --add-port=8888/tcp --permanent    #开启端口
firewall-cmd --list-ports    #查看开放的端口
firewall-cmd --query-port=8888/tcp    #查看端口是否开启
lsof -i:8888    #查看端口占用
netstat -tunlp |grep 8888    #查看指定端口的进程情况

最后是怎么解决的呢…

reboot重启下服务器就好了
