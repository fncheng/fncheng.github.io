yum install -y gcc-c++ make

yum install nodejs -y

npm install pm2 -g

期间遇到了一个问题:==yum安装软件竟然报错：rpmdb open failed==

解决办法

#首先清除掉缓存，之后再重建就可以了
rm -f /var/lib/rpm/__db*
rpm --rebuilddb
