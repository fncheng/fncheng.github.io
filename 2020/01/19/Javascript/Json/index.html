Json的表示及序列化和反序列化

发表于 2020-01-19 更新于 2020-03-02

Json(JavaScript对象表示法)

json的语法可以表示三种类型的值:

  • 简单值
  • 对象
  • 数组

1.简单值

json字符串和json对象

//字符串对象格式
{
    "name": "JSON",
    "address": "北京市西城区",
    "age": 25
}
//数组对象格式
[
    {
        "name": "JSON",
        "address": "北京市西城区",
        "age": 25
    },
    {
        "name": "JSON1",
        "address": "北京市东城区",
        "age": 28
    }
]

JSON序列化和反序列化

首先要搞懂什么是序列化?.

序列化（Serialization）是将对象的状态信息转换为可以存储或传输的形式的过程。

以json为例,简而言之就是将JavaScript对象以json格式存储到到*.json文件中；同理反序列化就是将.json文件中的json对象转换为JavaScript对象。

序列化——JSON.stringify()

var adress = {
    name: "JSON",
    address: "北京市西城区",
    age: 25
};
var result = JSON.stringify(adress, ["name", "age"]);
console.log(result);
// 输出结果 {"name":"JSON","age":25}

反序列化——JSON.parse()

var adress1 = JSON.parse(result);
console.log(adress1.address);
//输出结果  北京市西城区
